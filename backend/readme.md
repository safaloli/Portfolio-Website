# Backend Tech Stack
- Express for backend
- Neon for database management
- Sequelize orm with postgres language for database
- Git hub for verson controlling
- Joi for data validation


# Personal Portfolio Web App

## Installation
- Run `pnpm install` in terminal to install packages
    
## Backend Setup
- Run `pg_dump "your_neon_connection_string" > backup.sql` in terminal to import database from neon
